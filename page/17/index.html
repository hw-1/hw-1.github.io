macOS Sierra安装软件提示文件已损坏问题解决

macOS Sierra安装软件提示文件已损坏问题解决

macOS Sierra 以前：

打开系统偏好设置，选择安全性与隐私，通用 ，选择“任何来源”，然后点击允许来自任何来源，这样就不会出现“软件已损坏”或“不是Mac Appstore下载的”的提示了

macOS Sierra：

终端 输入 sudo spctl –master-disable 回车
输入你的开机密码 ，回到系统偏好设置的“安全与隐私”，勾选“允许任何来源”
